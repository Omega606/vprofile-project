# vprofile-project kubernetes deployment on AWS

Prerequisites

* JDK 1.8 or later
* Maven 3 or later
* MySQL 5.6 or later
* Apache Tomcat 8.5 or up to Tomcat 9
* kubectl 1.28.5 or later
* kops 1.28.4 or later
* Helm 3.14.3 or later

Technologies

* Spring MVC
* Spring Security
* Spring Data JPA
* Maven
* JSP
* MySQL
